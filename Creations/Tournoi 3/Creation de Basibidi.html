Captain Ability Multiplie l'ATK des personnages de type STR, DEX et QCK par 2,5 et les HP par 1,2. Si le coup spécial est utilisé, multiplie l'ATK du personnage par 3 pendant un tour.
Special (28→14 puis 13) Le personnage change de type ( STR , DEX ou QCK au choix) puis augmente les dégâts infligés sur certains ennemis selon le type choisi :
  • Si le type STR est conservé, multiplie les dégâts infligés aux personnages de type DEX par 1,3 pendant un tour.
  • Si le type DEX est choisi, multiplie les dégâts infligés aux personnages de type QCK par 1,3 pendant un tour.
  • Si le type QCK est choisi, multiplie les dégâts infligés aux personnages de type STR par 1,3 pendant un tour.
Une fois le type sélectionné, il est conservé jusqu'à la fin de la quête ou jusqu’au prochain lancement du coup spécial.
